applyXorCipher :: String -> String -> IO String Source #

Inline call to CPP function cn::applyXorCipher. Equivalent to HS function applyXorCipher. Used to encode or decode a message with XOR and key. For example,

>>> applyXorCipher "message" "my key"
"\NUL\FSS\CAN\EOT\RS\b"
>>> applyXorCipher "\NUL\FSS\CAN\EOT\RS\b" "my key"
"message"
